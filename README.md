# CURSOR-COURSE
This repository is about Cursor for the course "Topicos en Ingenieria de Software" 
En este repositorio se está desarrollando el curso "Tópicos en Ingeniería de Software" siguiendo los contenidos del curso sobre Cursor disponible en Santander Open Academy.

Hasta ahora, se ha creado una calculadora básica en Python que puede realizar las siguientes operaciones: suma, resta, multiplicación y división. El usuario puede seleccionar la operación y proporcionar los números, y el programa muestra el resultado correspondiente. Además, la calculadora maneja errores comunes como la introducción de valores no numéricos o la división por cero.

Este proyecto sirve para aplicar conceptos fundamentales del curso, como la estructuración de código en Python, el manejo de errores y la interacción con el usuario a través de la consola. El repositorio se irá actualizando conforme avancemos en los módulos del curso, integrando nuevas funcionalidades y buenas prácticas recomendadas en las lecciones.
